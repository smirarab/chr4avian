* Script for the main analyses showing quartet score and monophyly

* Check out [clade-analysis](clade-analysis) for scripts that perform the actual analyses. 

* 63K_trees.names_header.txt.xz
* all-clade.stat.xz: shows branch quartet score (BQS) for each group 
* all-monphyletic.txt.xz: shows the monophyletic genes for each group
* all-present.txt.xz: shows the number of taxa present from each group
* draw-movingaverage.r: Draw moving average based on the text files above.
