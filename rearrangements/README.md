See [../maf2synteny](../maf2synteny) directory for scripts used to obtain the synteny blocks used here. 

* draw.R: Draws the synteny plot (Figure 1B)
* figure1b.tsv: summary of results used in Figure 1B. 
* totab_chr4.csv.xz
* totab_CM030196.1_CICMAG.csv
* totab_CM030197.1_CICMAG.csv
* galGal4ToGalGal6.over.chain.gz: Chain file to connect galgal4 coordinated (used by B10K) to galgal6 coordinates
* outlier_plot.R: Draws the supplementary plot with outliers indicated for each genome. 			
* outlier-regions-by-maf2synteny.tsv: maf2synteny blocks, as depicted in the outlier plot
* outlier-regions-by-maf2synteny-summary.tsv: Rough coordinates of the outliers for each VGP genome
